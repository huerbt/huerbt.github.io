文档字符串常用特征

指令 描述
:param 参数描述
:arg 用来描述输入参数
:key 关键字描述
:type 可以使用另一种语法描述形参或实参类型。:param type param_name: description
:ivar或者:var 任何变量的描述。通常用于实例变量的描述
:vartype 变量类型描述
:py:meth: :py:func: :py:class: :py:attr: 用来交叉引用python函数、类、属性的方法例如。:py:meth:`MyClassA.method_a 反引号
