迭代人生的必要工具Git（01）

发表于 2019-09-09 | 分类于 用抖音费曼学习系列

今天准备开始拍一个系列的抖音短视频《用抖音费曼学习》，就是把自己掌握的知识用费曼学习的方法，在巩固一遍。

每次拍摄都需要规划一下，这里就是记录我思路的地方。

费曼学习法主要步骤有四步：

准备学习的概念

迭代、Git介绍

教给别人

属于陈述性知识，希望他们学会做某件事或使用某样东西。

  • 文档工具（MarkDown）
  • 命令行
  • Git
  • GitHub（推荐华为云）

逻辑依据

如果你的人生有下面三个诉求：

  1. 有一个词和“迭代”意思比较接近，叫“升级”，你是否需要升级自己
  2. 你是否需要持续在某个方向上持续升级
  3. 如果你升级的方向涉及多种知识，且知识想对复杂，难度较大

举例，以我自己来看，我需要学习量化投资——即利用数学和编程的方法对投资作出更合理的决策。

涉及了数学、python编程、机器学习、股票投资这些方面，每个方面涉及的知识点又特别多。我遇到了这样的问题：

  • 我想一下掌握是不可能、也需要避免学完一个，过一段时间又遗忘的问题
  • 我要建立一个程序系统，让它能自动运行复杂的数据获取，计算、展示数据、决策等，但是这个系统我不可能一下就建出来、需要一步一步来，而且过程中出现思考错误、执行错误也是常有的事，那我怎么管理呢怎么不让错误影响之前作对的事

绩效目标

明确告知学完课程后将获得哪些好处

  1. 自己的文档，方便日后查询
  2. 自己犯了错误，可以更容易更改
  3. 知识、技能的收益都可以在迭代中，一层一层累积起来，形成自己的知识库
  4. 可以和更优秀的人进行智力合作，共同成长

学习活动

什么是迭代

有一个词和“迭代”意思比较接近，叫“升级”，迭代为什么不直接叫升级，关键是迭代意味着你做事或学习的过程中，会不断试错，不断调整，不断精进，最后自然而然得到的一个结果，这个结果是升级，

迭代是通过不断重复让每次重复又比之前更好一点，为什么会更好一点，是因为你知道之前错在那了，你设想了新的路线，再次尝试也许这次也错了，但是你也是成长了，试过才知道就是这个道理。

你是否需要迭代

你的人生是否已经找到了一个方向，是你需要不断精进的？这个方向是否涉及的知识很多，而且比较难？如果回答都是“是的”，我相信你已经是超过很多人了。

那么你在学习或做事情的过程中，是否对要学习的知识数量、难度、以及自己犯的错误做过管理？

你过去放弃的事情是不是总是因为不知道自己哪里出错、不知道为什么一定要学这个知识久而久之最终放弃？

好吧你需要学习我这个系列的课程，使用各种工具帮助自己迭代成功。

你要先拥有第一个工具——Git

你今天只需要知道

  1. Git是一个可以让你每次在作出小的迭代时，都可以记录的工具，当你发现这一次迭代有错误需要返回上一次迭代时很容易的工具。
  2. Git是针对电脑上的文件做迭代管理的，所以你需要至少会在电脑上写文档，或者是编程文件。
  3. Git是在你本机电脑上做管理，但是和Git仓库配合，那就可以在你的多台电脑上对它们进行管理了。推荐GitHub，或者尤其是华为云的代码仓库。
